Frequently Asked Questions

Mohon di cek apakah koneksi saya bermasalah?
Hal hal yang harus di lakukan sbb :
1. Tanyakan no customer
2. Cari di datek
3. Tanyakan untuk layanan yang kmna yg bermasalah
4. Jika portnya merah ( berarti indikasi fisik down )
5. Cek perangkat sisi customer
6. Jika portnya hijau ( fisik hidup )
7. Klik sw untuk cek mac address ke dua sisi.
8. Terbaca informasikan ke customer untuk cek perangkat / configurasi di sisi customer
9. Tidak terbaca eskalasi L2
Mohon di cek koneksi ke cloud apakah ada masalah?
Dalam case ini cloud adalah ( GCP,ALIBABA,AWS,AZURE dll) Jika point A 1- 8 ok maka lanjut sbb :

GCP ( Google Cloud Platform )

Beberapa pertanyaan yang harus di tanyakan ke customer :
1 .Ping ptp nya reply atau tidak ( related ke point A nomor 7 )
2. BGP nya up kah.
3. Setting BGPnya boleh di infokah.
4. Jika semua sudah ok dan amsih down eskalasi L2

Alibabacloud

Beberapa pertanyaan yang harus di tanyakan ke customer :
1. Jika menggunakan BGP, apakah BGP nya sudah up? Setting BGP nya sudah benar?
2. Ping ptp nya reply kah?
3. Setting routing static nya sudah benar kah?
4. Jika semua sudah ok dan amsih down eskalasi L2
Mohon di cek BW saya ko ga sesuai?
1. Tanyakan no customer
2. Cari di datek
3. Tanyakan untuk layanan yang kmna yg bermasalah
4. Klik interface untuk cek BW.
5. Informasi bw ada di dalam kolom
6. Sesuai informasikan ke Customer
7. Tidak sesuai eskalasi L2
